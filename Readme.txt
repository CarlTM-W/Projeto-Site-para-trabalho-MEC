Requisitos:

	Python (versão 10 para cima);
	Django ( Versão mais atual possível)

Como iniciar o projeto:

 1° Deslocar o terminal para dentro da pasta do projeto;
 2° Dentro da pasta, digite o seguinte comando: (Windows) "Python manage.py runserver", (Linux) "python3 manage.py runserver".
 3° após a etapa 2, escreva em seu navegador: "localhost:8000"
 4° Pronto, o projeto esta rodando.
